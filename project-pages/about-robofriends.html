About the Project

The project, RoboFriends, is part of a Udemy course and was my first experience using the Javascript framework, React. In this project, we make a call to Json Placeholder's user API in order to populate each card component with a name and email. The user can then type in the search field in order to filter through the robots and find which ones have a name that matches what was entered.

While I cannot say that this project and the SmartBrain project are completely my own, this course exposed me to a lot of very important and useful tools for web development. This includes: How to use Git/Github, what React is and how you can use it, and a further understanding of API calls and how you can tie the front-end of a website to its back-end.

Note: Because this project is part of a code-along course, meaning that the code and ideas behind the project were not my own, I have decided to omit MY version of the project's github. You can however find the final code by instructor, Andrei Neagoie, here.

About the Project

The 'Portfolio' project is exactly how it sounds, a portfolio. This project in particular is a redesign of my original portfolio site, which can be seen here. While redesigning the website I had two goals in mind: giving each project their own about page rather than putting all the information on a card and providing easier access to viewing a project's code.

Because this project was on the simple side and requires very little updating, I opted to use traditional HTML and CSS rather than a Javascript framework like React. I also chose to use Tachyons to give myself more practice using a less-opinionated CSS framework.

About the Project

Food Order version 2 is the updated version of the Island Breeze Food Order Application. Both versions allow the user to add and remove items from their cart as well as change the quantity of each item in the cart. It calculates the cart total, displays it, and continually updates it as the user changes their selections.

One major thing that I learned while doing this project is that using a more opinionated CSS framework, such as Materialize and Bootstrap, isn't always easy or the best option for all applications. While it does speed up development, it can lead to your websites looking eerily similar to others, which isn't ideal for when you want your website to stand out. Of course you can then make alterations, however, that isn't always easy to do and can lead to unexpected bugs.

For more information on what changed and why, bugs I've encountered, and my ideas for improvement, please visit the project's GitHub.
